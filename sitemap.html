Связь с другими зонами
Wizard step 1 — теперь два таба: «Из библиотеки знаний» (выбор существующего) или «Загрузить новый» (документ автоматически попадает в КБ).

L-01
Библиотека курсов
/library
Все курсы workspace в одном месте. Видна всем участникам, включая Author. Главное отличие от «Мои курсы» — колонка автора и фильтры.

  • Фильтры по автору / статусу
  • Поиск по названию
  • Открытие → editor / экспорт
T-01
Команда
/team
Управление участниками workspace. Только для Owner/Admin. Лимит мест из тарифа.
  • Таблица: аватар, email, роль, статус, курсов, активность
  • Приглашение по email (модалка)
  • 3 роли: Owner · Admin · Author
  • Статусы: Active · Invited · Inactive
  • Изменение роли / деактивация
B-01
Биллинг
/billing
Текущий план, использование квот, способ оплаты, история счетов. Сравнение тарифов и переход.
  • Карточка текущего плана с usage-баром
  • Кол-во курсов / мест по тарифу
  • Способ оплаты (карта)
  • История счетов с PDF-скачиванием
  • 3 тарифа: Starter / Pro / Enterprise

06
Системные состояния
Универсальные паттерны — встречаются на нескольких экранах
Состояние
Где встречается
Поведение
Empty state
Дашборд (нет проектов)
Иллюстрация + текст «У вас пока нет курсов» + кнопка «Создать первый курс».
Loading
Парсинг файла · генерация AI · сборка SCORM
Прогресс-бар с понятным шагом процесса. Дисейбл UI, чтобы пользователь не дёргал.
Parse Error
W-01 «Загрузка документа»
Сканированный PDF без OCR, неподдерживаемый формат, превышение 50 МБ — конкретное сообщение и кнопка «Загрузить другой».
AI Error
W-03 «Прогресс генерации»
Таймаут AI или сбой после 2 retry. Сохраняется частичный результат + кнопка «Попробовать снова».
Export Error
E-04 «Экспорт»
Сбой сборки SCORM ZIP. Лог ошибки + контакт поддержки.
Unsaved warning
E-01 · E-02
Попытка уйти со страницы с несохранёнными правками — модальное предупреждение.
Large document
W-01 «Загрузка документа»
Документ >200 страниц — предупреждение о возможном времени и предложение обработать частями.
Export success
E-04 «Экспорт»
SCORM/XLSX готов. Auto-скачивание + сообщение «Файл готов» + инструкция загрузки в LMS.
404 / 500
Любой неизвестный URL · сбой сервера
Системная страница ошибки с возвратом на дашборд и контактом поддержки.

Методолог · T&D

Первичный пользователь · полный доступ в MVP

Создаёт курсы: загружает документы, настраивает параметры, редактирует черновик, экспортирует SCORM. В MVP — каждый методолог работает со своими проектами.

/projects /projects/new /projects/[id]/edit /projects/[id]/test /projects/[id]/export /account
🖥️

IT-администратор LMS

Вторичный пользователь · вне самой системы

Получает готовый SCORM ZIP от методолога и загружает в корпоративную LMS (Moodle, iSpring Learn, Эквио). Не работает в TalapLab напрямую — взаимодействует только с артефактом экспорта.

не входит в систему
🔧

Системный администратор

On-premise развёртывание · DevOps

Разворачивает TalapLab через Docker Compose на сервере клиента. Не имеет UI в MVP — управление через переменные окружения и логи.
